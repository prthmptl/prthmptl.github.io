## Overview

This portfolio site serves as my personal corner of the internet. It's where I showcase my projects, share my thoughts through blog posts, and document my learning journey.

## Features

- **Static Site Generation** - Fast page loads with pre-rendered HTML
- **Content Collections** - Type-safe markdown content management
- **Responsive Design** - Looks great on all devices
- **SEO Optimized** - Meta tags and structured data
- **Dark Mode Ready** - Easy to extend with theme support

## Technical Details

Built with Astro for its excellent developer experience and performance characteristics. Tailwind CSS handles styling with a utility-first approach.

### Key Decisions

1. **Astro over Next.js** - No client-side JavaScript needed for this content-focused site
2. **Markdown for content** - Easy to write and version control
3. **No CMS** - Git-based workflow keeps things simple

## Lessons Learned

- Content collections are incredibly powerful for type-safe content
- Tailwind's utility classes speed up development significantly
- Static sites can handle most portfolio/blog needs perfectly
